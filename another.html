Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerText helps to Gets or sets the text content of an element, InnerHTML Gets or sets the HTML content inside an element with html tags and textcontent , Gets or sets the text content of an element without any HTML tags.

Question-3: What is event delegation in the DOM?

It is particularly useful for handling events on elements that may be added or removed from the DOM dynamically.

Question-4: What is event bubbling in the DOM?

such as a mouse click or key press, "bubbles" up from the target element to its parent elements, all the way up to the root of the document .

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: document.createElement('tagName').
Add an element: parent.appendChild(element) .
Remove an element: element.remove() or parent.removeChild(element).
